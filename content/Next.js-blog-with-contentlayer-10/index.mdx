## About Page

about 페이지와 AboutCoverSection 컴포넌트를 생성 후 AboutCoverSection 컴포넌트를 about 페이지에 임포트 합니다.

```javascript title="src/app/about/page.js"
import AboutCoverSection from "@/src/components/About/AboutCoverSection";

const About = () => {
  return (
    <div>
      <AboutCoverSection />
    </div>
  );
};

export default About;
```

```javascript title="src/app/about/page.js"
import AboutCoverSection from "@/src/components/About/AboutCoverSection";
import Skills from "@/src/components/About/Skills";

const About = () => {
  return (
    <>
      <AboutCoverSection />
      <Skills />
      <h2 className="mt-8 font-semibold text-2xl self-start mx-20 text-dark ">
        Do you have a project idea? Contact me, and together we can turn that
        dream into reality.
      </h2>
    </>
  );
};

export default About;
```
