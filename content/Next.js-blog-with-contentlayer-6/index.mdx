src/components/Home/RecentPosts.js 를 생성합니다.

```javascript title="src/components/Home/RecentPosts.js"
import { sortBlogs } from "@/src/utils";
```

위 코드에서 sortedBlogs.slice 부분은 slice 메서드를 사용하여 sortedBlogs 배열에서 특정 범위의 요소를 선택하는 데 사용됩니다.

slice 메서드는 두 개의 인자를 받는데, 첫 번째 인자는 추출을 시작할 인덱스(이 경우는 4)이고, 두 번째 인자는 추출을 종료할 인덱스(이 경우는 10)입니다. 중요한 점은 slice 메서드는 첫 번째 인덱스는 포함하지만, 두 번째 인덱스는 포함하지 않는다는 것입니다.

즉, slice(4, 10)은 인덱스 4부터 9까지의 요소를 새로운 배열로 반환합니다.

HomeCoverSection에 가장 최신글(인덱스 0번), FeaturedPosts에 인덱스 1~3번, RecentPosts에는 인덱스 4~9번의 글 6개를 표현하기 위해 위처럼 작성했습니다.

각 인덱스는 추후에 개인에 원하는대로 수정하시면 됩니다.

⚠️ 실습을 위해 content폴더에 최소 10개의 컨텐츠(폴더/.mdx파일)가 있어야 합니다! 10개 까지 더 만들어주세요!

이제 `<article>`태그에 들어갈 컴포넌트를 만들어봅시다.

src/components/Blog/BlogLayoutThree.js 를 생성합니다.

BlogLayoutThree 컴포넌트는 BlogLayoutTwo 컴포넌트와 거의 동일하기 때문에 코드 전체를 복사해서 가져온 후 수정합니다.

```javascript title="src/components/Blog/BlogLayoutThree.js"
import { format } from "date-fns";
import Image from "next/image";
import Link from "next/link";
import React from "react";

const BlogLayoutThree = ({ blog }) => {
  return (
    <div className="group grid grid-cols-12 gap-4 items-center text-dark">
      <Link href={blog.url} className="col-span-4 rounded-xl overflow-hidden">
        <Image
          src={blog.image.filePath.replace("../public", "")}
          placeholder="blur"
          blurDataURL={blog.image.blurhashDataUrl}
          alt={blog.title}
          width={blog.image.width}
          height={blog.image.height}
          className=" aspect-square w-full h-full object-cover object-center group-hover:scale-105 transition-all ease duration-300"
        />
      </Link>
      <div className="col-span-8 w-full">
        <span className=" uppercase text-accent font-semibold text-sm">
          {blog.tags[0]}
        </span>
        <Link href={blog.url} className="inline-block my-1">
          <h2 className="font-semibold capitalize text-lg">
            <span className="bg-gradient-to-r from-accent/50 to-accent/50 bg-[length:0px_6px] group-hover:bg-[length:100%_6px] bg-left-bottom bg-no-repeat transition-[background-size] duration-500">
              {blog.title}
            </span>
          </h2>
        </Link>

        <div className="capitalize text-dark/50 font-semibold">
          {format(new Date(blog.publishedAt), "MMMM dd, yyyy")}
        </div>
      </div>
    </div>
  );
};

export default BlogLayoutThree;
```

코드 전체를 가져온 후 BlogLayoutThree로 이름변경 후 RecentPosts 컴포넌트에 임포트 합니다.
